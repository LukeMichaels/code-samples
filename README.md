# Code samples, Luke Michaels

Self-contained code samples written to accompany job applications. Each folder has its own README describing what it does, the problem it solves, and why I am proud of it.

- [`cta-block/`](./cta-block) is a PHP sample: a cache-safe, A/B-testable WordPress call-to-action block built on ACF.
- [`accessible-modal/`](./accessible-modal) is a JavaScript sample: a dependency-free accessible modal built on the native `<dialog>` element.
- [`campaign-migration-cli/`](./campaign-migration-cli) is a PHP sample: a WP-CLI command that migrates legacy campaign action records into WordPress without duplicating on rerun.

Nearly all of the code I've written professionally lives inside of private repos. These samples reproduce patterns from work I have shipped, rewritten here as clean, self-contained examples without any proprietary code.
